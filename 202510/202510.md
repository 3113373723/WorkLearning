# 学习笔记
## 统计仓库代码量
网上的方法统计的不太准确
```
git ls-files | xargs wc -l | awk '{print sum+=$1}; END {print sum}'
```
推荐使用
```
git log --pretty=format: --numstat|awk '{add += $1; subs += $2; loc += $1-$2} END {printf "added lines:%s, removed lines:%s, total lines: %s\n", add, subs, loc}'
```
